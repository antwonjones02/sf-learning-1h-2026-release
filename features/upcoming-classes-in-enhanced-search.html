Upcoming Classes in Enhanced Search

Feature #42 of 54

What Is This Feature?

Enhanced search results now include upcoming class schedules, allowing learners to find and register for classes more easily.

What Is Changing?

Class schedule information is surfaced directly in search results.

Why Should We Care?

Learners can find and register for classes without leaving search results. Reduces drop-off in the registration process.

Risk of Inaction

Learners must navigate to individual class pages to see schedules — more friction.
