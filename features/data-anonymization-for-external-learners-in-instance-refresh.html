Data Anonymization for External Learners in Instance Refresh

Feature #28 of 54

What Is This Feature?

Administrators can now anonymize Learning data for external learners during instance refresh operations to protect personally identifiable information in non-production instances.

What Is Changing?

The Instance Refresh Tool now includes data anonymization specifically for external learner data in Learning.

Why Should We Care?

Strengthens data privacy compliance (GDPR, CCPA) by ensuring PII is not exposed in test environments.

Risk of Inaction

External learner PII may persist in non-production instances, creating compliance and privacy exposure.
