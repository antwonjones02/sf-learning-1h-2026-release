New Manage Super Admin Access Tool

Feature #9 of 54

What Is This Feature?

Administrators can now manage super admins for both Learning and Platform from a unified access management page. The old tool is split into two: Manage Super Admin Access and Manage RBP Admin Access.

What Is Changing?

A single tool replaces the combined admin tool. Learning super admin management is now available alongside Platform.

Why Should We Care?

Simplifies security administration and provides clearer separation between super admin and RBP admin functions.

Risk of Inaction

No action needed — automatically applied. Admins should familiarize themselves with the new tool location.

Recommendation Train admin team
