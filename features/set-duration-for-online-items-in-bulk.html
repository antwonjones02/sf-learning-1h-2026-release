Set Duration for Online Items in Bulk

Feature #24 of 54

What Is This Feature?

Administrators can now add or update duration for multiple online learning items at once through the Item Connector.

What Is Changing?

Bulk duration updates via Item Connector — no more updating each item individually.

Why Should We Care?

Saves significant admin time when managing large content libraries. Ensures duration data is consistent for reporting.

Risk of Inaction

Duration updates remain one-at-a-time. Heavy administrative burden for large catalogs.
