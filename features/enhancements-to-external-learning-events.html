Enhancements to External Learning Events

Feature #21 of 54

What Is This Feature?

New fields have been added to the External Event entity, allowing learners, managers, and administrators to capture more comprehensive information when logging external learning.

What Is Changing?

External learning records now support additional data fields — more context about external training, conferences, and certifications.

Why Should We Care?

Creates a more complete picture of employee development, including learning that happens outside the LMS.

Risk of Inaction

External learning records remain basic. Organization loses visibility into the full scope of employee development.
