Credit Hours in Open Content Network OData API

Feature #23 of 54

What Is This Feature?

The OCN OData API now supports creditHours when posting OCN items to SAP SuccessFactors Learning.

What Is Changing?

Credit hour values from content providers can now flow into Learning automatically via the API.

Why Should We Care?

Enables accurate credit tracking for professional development and continuing education requirements.

Risk of Inaction

Credit hours must be manually entered for OCN content. Time-consuming and error-prone.

Recommendation Enable if using credit hours
Type Changed
Impact Minor
