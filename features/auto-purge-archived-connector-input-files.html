Auto-Purge Archived Connector Input Files

Feature #36 of 54

What Is This Feature?

Archived connector input files are now automatically purged after a defined retention period.

What Is Changing?

Old connector files no longer accumulate indefinitely. The system automatically cleans up archived files.

Why Should We Care?

Reduces storage bloat and simplifies system maintenance. Keeps the connector file area manageable.

Risk of Inaction

Archived files continue to pile up, consuming storage and making troubleshooting harder.
